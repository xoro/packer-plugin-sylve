# Sylve VM Builder

Type: `sylve-vm`
Artifact BuilderId: `xoro.sylvevm`

~> **Note**: The `sylve-vm` builder is planned for a future release and is
not yet implemented. See the [`sylve-iso`][docs-sylve-iso] builder for the
currently available ISO-based builder.

The `sylve-vm` builder will start from an existing Sylve VM snapshot instead
of booting from an ISO image, run Packer provisioners over SSH, and produce
an updated artifact.

[docs-sylve-iso]: sylve-iso.mdx
